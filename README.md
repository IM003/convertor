# XML ⇄ JSON Converter (JavaFX)

## Vidéo 5min
https://youtu.be/tgMfD3pPl4Y?si=dbXbYBz9KJAuvJIl

## Description
Ce projet est une application Java permettant de convertir des données :
- XML vers JSON
- JSON vers XML

L’application propose deux modes :
- Conversion avec API
- Conversion sans API (implémentation manuelle)

Une interface graphique JavaFX est utilisée pour faciliter l’utilisation.

## Technologies utilisées
- Java JDK
- JavaFX
- IntelliJ IDEA
- Bibliothèque org.json (uniquement pour la partie API)

## Structure du projet
- Main.java : interface graphique JavaFX
- ApiConvertor.java : conversion avec API
- ManualConvertor.java : conversion sans API

## Exécution
1. Lancer la classe `Main`
2. Coller du XML ou du JSON dans la zone correspondante
3. Cliquer sur le bouton de conversion souhaité
